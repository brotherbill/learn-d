# Real Number

A **real number** is any number that can be placed on a continuous number line.  
Real numbers include **all rational numbers** and **all irrational numbers**.

Real numbers cover:
- whole numbers  
- fractions  
- repeating decimals  
- terminating decimals  
- non‑repeating decimals  
- square roots  
- π, e, and other irrationals  

If a number can be located on the number line, it is a **real number**.

---

## 1. Real numbers include rationals and irrationals

### Rational numbers  
Numbers that can be written as a fraction of two integers:

```
a / b   (b ≠ 0)
```

Examples:
```
1/2
3/4
−5/3
10/1
```

### Irrational numbers  
Numbers that **cannot** be written as a fraction of integers.

Examples:
```
√2
π
e
```

Their decimals never end and never repeat.

---

## 2. The real number line

The real number line is a **continuous** line with no gaps.  
Every point on the line corresponds to exactly one real number.

```
… −3, −2.5, −1, 0, 0.333…, 1, √2, 3.14…, 5, …
```

Integers and rational numbers appear at specific points.  
Irrational numbers fill all the “in‑between” points.

The real number line is infinite in both directions and contains infinitely many numbers between any two points.

---

## 3. Real numbers vs integers

| Kind         | Definition                                 |
|--------------|---------------------------------------------|
| Integer      | whole number (…, −2, −1, 0, 1, 2, …)        |
| Real number  | any number on the number line               |

Every integer is a real number.  
Not every real number is an integer.

Examples:
```
3     → integer and real
3.5   → real, not integer
√2    → real, not integer
```

---

## 4. Real numbers vs rational numbers

| Kind            | Definition                                  |
|-----------------|----------------------------------------------|
| Rational number | integer divided by non‑zero integer          |
| Real number     | rational or irrational                       |

Examples:
```
1/3   → rational → real
0.25  → rational → real
√2    → irrational → real
π     → irrational → real
```

Real numbers are the larger set.

---

## 5. Decimal behavior

Real numbers can have:

- terminating decimals  
- repeating decimals  
- non‑terminating, non‑repeating decimals  

Examples:
```
0.5        → terminating
0.3333…    → repeating
√2 = 1.4142… (never repeats)
π  = 3.14159… (never repeats)
```

If a decimal never repeats and never ends, the number is irrational — but still real.

---

## 6. Real numbers are exact (in mathematics)

In pure mathematics, real numbers are **exact** objects.

- π is exactly π  
- √2 is exactly √2  
- 1/3 is exactly 1/3  

Real numbers are not approximations.  
They are ideal mathematical quantities that exist on the continuous number line.

(Computers approximate real numbers, but that belongs in floating‑point.md, not here.)

---

## 7. Why real numbers matter

Real numbers are essential for:

- measurement  
- geometry  
- physics  
- calculus  
- continuous quantities  
- mathematical analysis  

Real numbers let us describe values that are not whole and not fractional in a simple way.
