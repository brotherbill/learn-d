# Odd

An **odd** number is an integer that is **not** divisible by **2**.  
When you divide an odd number by 2, there is always a **remainder of 1**.

Example:
```
7 ÷ 2 = 3 remainder 1
```

So **7 is odd**.

Example of a non‑odd number:
```
8 ÷ 2 = 4 remainder 0
```

So **8 is not odd**.

---

## 1. Formal definition

An integer **n** is odd if there exists an integer **k** such that:

```
n = 2 × k + 1
```

Examples:
- 1 = 2 × 0 + 1  
- 7 = 2 × 3 + 1  
- −5 = 2 × (−3) + 1

Oddness applies to **all integers**, positive and negative.

---

## 2. Odd numbers are “one more than a multiple of 2”

Every odd number is exactly **one more** than a multiple of 2:

```
…, −5, −3, −1, 1, 3, 5, …
```

These are the integers you land on when you count by twos starting at 1.

---

## 3. Odd vs. even

- **Even** → divisible by 2  
- **Odd** → not divisible by 2  

Odd numbers always leave remainder 1 when divided by 2.

Examples:
```
7 ÷ 2 = 3 remainder 1   → odd
8 ÷ 2 = 4 remainder 0   → even
```

See **glossary/even.md**.

---

## 4. Why oddness matters

Odd numbers are important for:

- parity checks  
- alternating patterns  
- loop stepping  
- modular arithmetic  
- correctness in algorithms  

Oddness is one of the simplest and most useful arithmetic classifications.
